# Sistema Bancário em C++ (Qt Creator)

Este é um projeto de sistema bancário desenvolvido como parte do trabalho da UFSC. O sistema utiliza o framework Qt para a interface gráfica e a linguagem **C++** para a lógica.

## Funcionalidades
* **Cadastro e Login:** Criação e acesso à conta bancária.
* **Operações Financeiras:** Realização de saques, depósitos e transferências.
* **Consulta de Saldo:** Exibição do saldo atualizado em tempo real e extrato de mevimentações da conta.
* **Interface Gráfica:** Desenvolvida com Qt Widgets.

## Tecnologias Utilizadas
* **Linguagem:** C++
* **Framework:** Qt 6.10.0
* **IDE:** Qt Creator
* **Compilador:** MinGW / MSVC
